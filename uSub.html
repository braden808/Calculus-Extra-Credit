Integration


Integration is a key concept in Calculus. Many Calculus problems involve integration. Right now we will learn to find the integral using u-substitution.


Why u-substitution is preffered

U-substitution is preffered for usage because it automatically compensates for the values needs to be added when you solve for du. For other methods like the baby prime method, you will have to compensate manually.

Components and Tips


  • Make sure you have space for side work
  • In your sidework, identify what u will be and what du will be
  • Remember to add + C
  • Return to initial variable at the very end
