Integration


Integration is a key concept in Calculus. Many Calculus problems involve integration. Right now we will learn to find the integral using the Power Rule in integration form.


Rules and Tips


Power Rule

  • c stands for constant
  • n can never equal to -1
  • Remember to add + C

Sub Rules
  • Multiplication by constant: ∫cf(x) dx = c∫f(x) dx
  • Sum Rule: ∫(f + g) dx = ∫f dx + ∫g dx
  • Difference Rule: ∫(f - g) dx = ∫f dx - ∫g dx
